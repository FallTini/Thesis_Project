# Skin Cancer Detection (Deep Learning)

This repository contains my research project on **skin lesion classification (benign vs malignant)** using deep learning Models, developed as part of my thesis work.

## Objective
- Build a reliable model for **skin cancer detection**
- Focus on **Sensitivity / Recall (True Positive Rate)** due to the importance of detecting malignant cases
- Handle **severe class imbalance** in the dataset

## Dataset
The dataset is highly imbalanced:
- Benign: ~40,000 images  
- Malignant: ~392 images  

> The dataset is not included in this repository.
